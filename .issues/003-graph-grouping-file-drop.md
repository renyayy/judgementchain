# Geminiのグルーピングでファイルが漏れる

## 問題

Phase 2 のグルーピングで、Gemini が **すべてのファイルをグループに割り振るとは限らない**。
プロンプトで指示しても、LLMのレスポンスからファイルが抜け落ちることがある。
グループに割り振られなかったファイルは孤立ノードになるか、グラフに表示されない。

## 影響

- 一部のファイルがグラフに表示されない（ユーザーからの報告あり）
- グラフの網羅性が保証されない

## 該当コード

- `src-tauri/src/commands.rs` — Phase 2 グルーピングプロンプト＋レスポンスパース
